* <a href="https://vulcan9.github.io/taskml.doc/?href=/taskml.doc/doc/0.1.x/index" target="_blank">0.1.x</a>
* <a href="https://vulcan9.github.io/taskml.doc/?href=/taskml.doc/doc/0.2.x/index" target="_blank">0.2.x</a>


> v 0.1.4 버전부터 `<as-task>`태그 대신 `<task>` 태그를 사용할 수 있습니다.  
> v 0.2.0 버전부터 `preload`, `include` task 기능으로 외부 taskml 파일을 불러올 수 있습니다.
